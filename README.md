# Task Management Application

A full-stack task management application that helps users organize work, collaborate in teams, assign tasks, track progress, and communicate through comments and attachments.

## Features

- User registration and login
- Secure authentication and authorization
- Create, update, and manage personal tasks
- Create teams and manage team collaboration
- Assign tasks to specific members within a team
- Add comments under tasks for communication and updates
- Upload text file attachments to tasks
- Update user profile details
- Change account password
- Team-based task tracking and collaboration

## Modules

### Authentication
- Register a new account
- Login with existing credentials
- Secure JWT-based authentication
- Change password

### User Management
- View and edit user profile details
- Maintain account information securely

### Task Management
- Create tasks
- Update task details
- Assign tasks
- Track task progress
- Comment on tasks
- Attach text file attachments to tasks

### Team Management
- Create teams
- Add and manage team members
- Create tasks under a team
- Assign team tasks to a specific person in the group

## Project Workflow

1. User registers and logs in to the application.
2. User can create and manage personal tasks.
3. User can create a team and collaborate with team members.
4. Team tasks can be assigned to a specific member.
5. Members can comment under tasks for discussion.
6. Text file attachments can be uploaded to tasks.
7. Users can update their details and change their password.
